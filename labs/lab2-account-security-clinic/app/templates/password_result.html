Teaching prompts

Prompt 1: What password rules increase security the most (length, uniqueness, MFA) versus which rules mostly increase frustration?

Prompt 3: How would you teach “unique passwords” without shaming students who reuse them?
Quick rubric idea: - Strong: long (16+), unique, not predictable, stored safely - Better: long, unique, but stored unsafely (notes app / screenshot) - Risky: short, reused, patterned ("Spring2026!")
